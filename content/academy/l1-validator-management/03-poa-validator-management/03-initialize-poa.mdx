The `ValidatorManager` contract has some settings we can initialize:

```solidity
struct ValidatorManagerSettings {
    address admin; // The address of the admin (EoA, multisig, timelock, etc.)
    bytes32 subnetID; // The ID of the Subnet
    uint64 churnPeriodSeconds; // The number of seconds between churns
    uint8 maximumChurnPercentage; // The maximum percentage of validators that can be churned per churn period
}
```
