### Creating a Genesis for an EVM L1

The next step is to create a genesis block for your new blockchain. 

The genesis determines the starting state of your blockchain. This step is important because it determines configuration parameters such as:
- Chain ID
- Gas Limits
- Block Time
- Initial Token Allocations
- Enabled Precompiles

You can read more about the `genesis.json` file for `subnet-evm` [here](/docs/avalanche-l1s/upgrade/customize-avalanche-l1#chain-config).
